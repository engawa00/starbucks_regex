# StarbucksJP-Drink-Menu-Regex

日本のスターバックスのレギュラードリンクメニュー（各種カスタマイズを含む）を解析するための正規表現を提供するPythonプロジェクトです。

## 特徴
- サイズ、デカフェ等の変更、ミルクの変更、シロップやソースの追加・変更など、複雑なスターバックスの注文文字列にマッチする正規表現（`STARBUCKS_JP_DRINK_MENU_REGEX`）を提供します。
- `tkinter` を使用したシンプルなGUI判定アプリを同梱しており、手軽に注文文字列のテストが可能です。
- `pytest` を用いたテストコードを同梱し、継続的インテグレーション(GitHub Actions)にも対応しています。

## 動作要件
- Python 3.10 以上 (推奨)
- 標準ライブラリ (`re`, `tkinter`)
- `pytest` (テスト実行時のみ)

## 必要なライブラリのインストール
本プロジェクトのメイン機能とGUIアプリはPython標準ライブラリのみで動作するため、追加のインストールは不要ですが、テストを実行する場合は以下を実行してください。

```bash
pip install pytest
```

## 使い方

### 1. GUIモード (判定アプリの起動)
同梱の `app.py` を実行することで、GUIアプリが起動します。

```bash
python app.py
```

起動後、テキストボックスに注文文字列（例: `トール エクストラホイップ ダークモカチップフラペチーノ`）を入力して「判定する」ボタンを押すと、マッチ結果とグループ分けの詳細が表示されます。

### 2. Pythonスクリプトからの利用 (CLI / モジュール利用)
`starbucks_regex.py` をインポートして、独自のスクリプトから利用できます。

```python
from starbucks_regex import STARBUCKS_JP_DRINK_MENU_REGEX, match_drink_order, extract_drink_order

order = "グランデ オーツミルク変更 キャラメル マキアート エクストラホット"

# 完全一致するか判定
if match_drink_order(order):
    print("注文として妥当です！")

# 文章の中から注文部分を抽出
text = "今日のランチは トール スターバックス ラテ ホット を飲みました"
drink = extract_drink_order(text)
print(f"注文内容: {drink}")
```

## 開発者向け (テストの実行)
正規表現の変更や機能追加を行った場合は、`pytest` を用いてテストを実行し、既存のパターンが壊れていないか確認してください。

```bash
pytest tests/test_starbucks_regex.py
```
テストがすべてパス（緑色）になれば正常です。
