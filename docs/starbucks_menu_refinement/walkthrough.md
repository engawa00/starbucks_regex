- `.*` による広範なマッチングを廃止し、以下のプレフィックスを必須としました。
- 各単語間に `\s?` を挿入し、スペースの有無を許容するようにしました。

### tests/test_starbucks_regex.py

- 上記メニュー名のバリエーション（スペースあり・なし）をテストケースに追加しました。
- 以下の異常系テストを追加しました：
    - メニューにないプレフィックス（例：「牛丼 ティー ラテ」）
    - 単体でのマッチ（「フラペチーノ」「ティー ラテ」）※これらは正規のメニュー名ではないため
    - 期間限定メニュー（例：「ジョイフルメドレー ティー ラテ」）
